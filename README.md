## Project setup
```
npm install
```

## Vue
### Compiles and hot-reloads for development
```
npm run serve
```
### Compiles and minifies for production
```
npm run build
```
### Lints and fixes files
```
npm run lint
```


## Storybook
### Compiles storybook for development
```
npm run storybook
```
### Compiles and minifies for production
```
npm run build-storybook
```
